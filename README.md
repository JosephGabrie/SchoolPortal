# SchoolPortal
This is a school portal for Mountain View School, San Pedro Sula, Honduras.

It uses Svelte and Convex for high speeds and great performance. 
It has about 200+ Daily Active Users.

In order to run this simply run
1. npm install
2. npx convex dev
3. npm run dev
P.S. Make sure that you input your own workos and convex api key
